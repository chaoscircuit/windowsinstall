Chaos Connect Windows Installer
===============================

This is the installer for the Chaos Connect PC software. It installs
the GUI application as well as necessary drivers.

It can be built using NSIS 2.46. It may also compatible with other
verions, but it has only been tested with 2.46. In order for the
installer to work properly, it expects the newest ChaosConnect binary to
be in the chaosconnect directory.
